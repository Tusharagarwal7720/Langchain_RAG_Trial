#Smart Employee Training Assistant (RAG with Local LLM)

A desktop-based AI assistant for employee training and knowledge management. Users can upload documents, ask questions, and receive answers generated by a local LLaMA2 model using LangChain and Chroma vector store for retrieval-augmented generation (RAG).

##Features

Upload PDF, TXT, or DOCX documents and create a vector store for retrieval.

Ask questions and get AI-generated answers from uploaded documents.

View chat history or clear current conversation.

Delete all documents and vector store with confirmation.

Works fully offline using a local LLaMA2 model via langchain-ollama.

##Technology Stack

Python 3.11+

LangChain 0.2.16 — for chain management and RAG.

langchain-ollama 0.1.2 — local LLaMA2 model integration.

Chroma 0.4.24 — vector database for document embeddings.

SentenceTransformers — embeddings for document indexing.

Flet — for building a simple GUI interface.

Torch >= 2.0.0 — backend for LLaMA2 and embeddings.
